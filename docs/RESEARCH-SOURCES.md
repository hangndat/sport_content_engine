---

## 7. Cách thêm vào hệ thống

1. Chạy **Ingest** test với URL mới để kiểm tra RSS hợp lệ.
2. Vào **Admin → Nguồn tin** thêm nguồn thủ công (ID, type=rss, tier, url).
3. Hoặc thêm vào `src/config/sources.ts` và chạy `npm run db:seed` (nếu seed ghi đè).
